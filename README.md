# Packet Tracer - Connect to a Web Server
A Cisco Packet Tracer lab to observe how packets are sent across the Internet using IP addresses.

## Objectives
- Verify connectivity to the web server.
- Connect to the web server via the web client.

## Tools Used
- Cisco Packet Tracer

## Steps / Lab Procedure

### **Part 1: Verify Connectivity to the Web Server**

### **Part 2: Connect to the Web Server via the Web Client**
1. On PC0, select **Desktop > Web Browser**.
2. Enter `172.33.100.50` in the URL field.
3. Click **Go** to connect and display the webpage hosted on the web server.

## Results
- Successful pings confirm end-to-end network connectivity.
- The web page successfully loads via the web browser using the IP address.
